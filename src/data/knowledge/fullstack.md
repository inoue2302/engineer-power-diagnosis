# フルスタック開発トレンド 2025-2026

## フレームワーク
- Next.js App Router + Server Actions でフルスタック TypeScript
- Remix / React Router v7 でのプログレッシブエンハンスメント
- SvelteKit でフルスタック Svelte 開発
- Hono + htmx で軽量フルスタック構成

## フルスタック TypeScript
- tRPC / Hono RPC でクライアント〜サーバーの型安全を担保
- Drizzle ORM / Prisma でDB〜APIの型を一貫管理
- Zod スキーマをフォームバリデーション〜API入力検証で共有
- モノレポ（Turborepo / Nx）で型定義を一元管理

## AI 統合
- Vercel AI SDK で LLM をフロントエンドに統合
- LangChain / LlamaIndex で RAG パイプライン構築
- AI コーディングアシスタント（Cursor / GitHub Copilot）の活用が前提スキルに
- プロンプトエンジニアリングが開発者の基本スキルに

## デプロイ・ホスティング
- Vercel / Netlify / Cloudflare Pages でゼロコンフィグデプロイ
- Supabase / Neon でサーバーレス PostgreSQL
- Upstash で Redis / Vector DB / Kafka をサーバーレスに

## 設計スキル
- ドメイン駆動設計（DDD）の基礎理解
- クリーンアーキテクチャの実践的な適用
- API ファースト設計（OpenAPI / GraphQL schema first）
